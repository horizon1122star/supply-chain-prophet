Interactive explainer for the Weather Agent in Supply Chain Prophet — covers APIs, data flow, thresholds, and shipping lane logic

What the weather agent does
1
Receives company + scenario
Orchestrator calls the agent with e.g. "Apple Inc." + "Taiwan earthquake". The agent ignores the company name — its job is purely geographic. It maps the scenario to a set of shipping lanes that matter.
2
Queries Open-Meteo for each lane
Sends one async HTTP request per shipping lane coordinate (lat/lon). Asks for 7-day forecast: max wind speed, max precipitation, wave height. No API key. No rate limit. Completely free.
3
Evaluates each lane against thresholds
Wind > 60 km/h → critical. Wind 40–60 → elevated. Wave height > 4m → elevated. Precipitation > 50mm → monitor. Each lane gets a severity level and a reason string.
4
Streams log lines to the UI
As each lane resolves, yields a log line immediately via SSE. The agent panel lights up lane by lane — judges see it thinking in real time. Deliberate 250ms delay between lines for demo effect.
5
Emits structured result
Returns risky_lanes list + lane_details dict with exact wind/wave values. Orchestrator uses this to color shipping lanes on the map and add a weather signal to the probability score.

Primary API
Open-Meteo Fully free · no key
Open-source weather API. Covers marine weather, wave height, wind speed, precipitation. Unlimited requests. No account needed. Updated hourly. This is your primary source — use it for everything.
https://api.open-meteo.com/v1/forecast?latitude={lat}&longitude={lon}&daily=windspeed_10m_max,precipitation_sum&forecast_days=7
Open-Meteo Marine API Fully free · no key
Separate endpoint from the same provider. Gives wave height, swell period, ocean current direction. Use this alongside the main forecast endpoint for shipping-lane accuracy. Critical for detecting typhoon swell.
https://marine-api.open-meteo.com/v1/marine?latitude={lat}&longitude={lon}&daily=wave_height_max,swell_wave_height_max
Backup API (if Open-Meteo is down)
Open Weather Map Free tier · key required
1,000 free calls/day. Good fallback for wind and precipitation. Doesn't have marine-specific wave data but useful as a second opinion. Free key from openweathermap.org takes 30 seconds to get.
https://api.openweathermap.org/data/2.5/forecast?lat={lat}&lon={lon}&appid={OWM_KEY}
NOAA / NHC (typhoon tracking) Fully free · no key
National Hurricane Center RSS/JSON feed. Use this specifically to detect named storms in the Pacific and Indian Ocean. Parse the active storm list and check if any track intersects your shipping lanes. This is your typhoon early warning.
https://www.nhc.noaa.gov/CurrentStorms.json
Hackathon note
For demo day, Open-Meteo is all you need. It's free, no key, no rate limit. Add OWM as a fallback if you have time. NOAA typhoon tracking is the bonus feature that makes judges nod — adds maybe 45 minutes to implement and looks very impressive. Prioritize only after the core 3 agents are stable.

Threshold logic — when does a lane become "risky"?
Wind speed
< 40 km/h Clear — normal ops
40–60 km/h Elevated — monitor closely
> 60 km/h Critical — vessel delays likely
Wave height
< 2.5m Clear
2.5–4m Elevated — heavy cargo affected
> 4m Critical — most vessels restricted
Precipitation
< 25mm/day Clear
25–50mm Monitor — port loading slows
> 50mm Elevated — flood risk at port
Named storm nearby
Any category Immediately critical — override all other signals
How severity maps to the orchestrator
The weather agent contributes a weather_risk_score (0–1.0) to the orchestrator. It's calculated as:

score = (critical_lanes × 0.4) + (elevated_lanes × 0.2)

The orchestrator Claude prompt receives this score alongside the other agents' outputs and weighs it into the final probability. Weather alone rarely exceeds 30% weight — it amplifies financial and news signals rather than standing alone.
Full weather.py
import httpx, asyncio, os
from agents.base import BaseAgent
from fallbacks import WEATHER_FALLBACK

SHIPPING_LANES = {
  "Taiwan Strait": (24.5, 119.5),
  "Red Sea": (12.5, 43.3),
  "Strait of Malacca": (2.5, 102.0),
  "Suez Canal": (30.5, 32.3),
  "South China Sea": (15.0, 114.0),
  "Cape of Good Hope": (-34.3, 18.5),
  "Strait of Hormuz": (26.5, 56.3),
}

def severity(wind, wave, rain):
  if wind > 60 or wave > 4.0:
    return "critical"
  if wind > 40 or wave > 2.5 or rain > 50:
    return "elevated"
  return "clear"

class WeatherAgent(BaseAgent):
  name = "weather"

  async def _execute(self, company, scenario):
    yield self._event("status", "scanning",
      log="▸ Checking 7 shipping lanes...")

    risky, details = [], {}

    async with httpx.AsyncClient() as client:
      for lane, (lat, lon) in SHIPPING_LANES.items():
        # fetch forecast + marine in parallel
        wx, mx = await asyncio.gather(
          client.get("https://api.open-meteo.com/v1/forecast",
            params={"latitude":lat,"longitude":lon,
              "daily":"windspeed_10m_max,precipitation_sum",
              "forecast_days":7,"timezone":"UTC"},timeout=8),
          client.get("https://marine-api.open-meteo.com/v1/marine",
            params={"latitude":lat,"longitude":lon,
              "daily":"wave_height_max",
              "forecast_days":7,"timezone":"UTC"},timeout=8),
        )

        w = wx.json()["daily"]
        m = mx.json().get("daily", {})
        wind = max(w.get("windspeed_10m_max",[0]))
        rain = max(w.get("precipitation_sum",[0]))
        wave = max(m.get("wave_height_max",[0]))

        sev = severity(wind, wave, rain)
        details[lane] = {"wind":wind,"wave":wave,"rain":rain,"severity":sev,"lat":lat,"lon":lon}

        if sev != "clear": risky.append(lane)
        icon = "▸" if sev=="clear" else "⚠"
        yield self._event("log", "live",
          log=f"{icon} {lane}: {wind:.0f}km/h wind · {wave:.1f}m swell")
        await asyncio.sleep(0.25) # pacing for demo

    yield self._event("result", "done",
      log=f"▸ {len(risky)} lanes at risk of {len(SHIPPING_LANES)} checked",
      data={"risky_lanes":risky,"lane_details":details})

  async def _fallback(self, company, scenario):
    for line in WEATHER_FALLBACK.get(scenario, WEATHER_FALLBACK["default"]):
      yield self._event("log", "live", log=line)
      await asyncio.sleep(0.3)
    yield self._event("result", "done",
      log="▸ Weather checked (cached)",
      data={"risky_lanes":["Taiwan Strait"],"lane_details":{}})
